# Spline Motion Warping - Demo Project

Demo project for the **Spline Motion Warping** plugin for Unreal Engine 5.6+.

📖 [Full Documentation](https://github.com/StaffanJOlsson/SplineMotionWarping-docs) | 🎬 [Tutorial](https://youtu.be/5JiHij93gV4)

## Requirements

- Unreal Engine 5.6+
- [Spline Motion Warping](https://www.fab.com/portal/listings/13abea64-26ff-4e61-b731-0b389d2e5738) plugin (install to the `Plugins/` folder)

## Setup

1. Clone this repository
2. Create a `Plugins/` folder and install the Spline Motion Warping plugin into it
3. Right-click `SplineMWDemo.uproject` → Generate Visual Studio project files
4. Build the project from source (Build → Build Solution in VS, or use UnrealBuildTool)
5. Open the project in Unreal Engine
